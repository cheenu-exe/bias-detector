# BiasGuard - AI Bias Detection Tool

An expert AI Auditor for Black Box Bias Detection and Algorithmic Fairness.

## Features
- **Historical Bias Audit**: Calculate Disparate Impact, Statistical Parity, and flip rates.
- **Real-Time Screening**: Detect data drift and sensitive proxies in live streams.
- **Interactive Dashboard**: Modern web UI for audting and monitoring.
